API Docs

Integrasikan uploader ini ke Node.js, bot WhatsApp, atau aplikasi apapun.

Base URL

Ganti https://your-project.vercel.app dengan domain Vercel kamu setelah deploy.

Base URL: https://your-project.vercel.app
POST /api/upload Upload satu atau banyak file

Headers (opsional)

HeaderValueKeterangan
x-api-keystringJika UPLOAD_API_KEY di-set di Vercel env

Body

FieldTypeKeterangan
filemultipart/form-dataFile yang di-upload (bisa multiple)

Response sukses

{
  "success": true,
  "url": "https://blob.vercel-storage.com/xxx.jpg",
  "filename": "photo.jpg",
  "size": 204800,
  "type": "image/jpeg",
  "uploadedAt": "2024-01-01T12:00:00.000Z"
}

Contoh — fetch (Node.js / Browser)

const FormData = require('form-data');
const fs = require('fs');
const fetch = require('node-fetch'); // npm i node-fetch

const form = new FormData();
form.append('file', fs.createReadStream('./image.jpg'));

const res = await fetch('https://your-project.vercel.app/api/upload', {
  method: 'POST',
  body: form,
  headers: form.getHeaders(), // auto set multipart boundary
});

const data = await res.json();
console.log(data.url); // https://blob.vercel-storage.com/xxx.jpg

Contoh — Bot WhatsApp (Baileys / WA-Multi-Device)

// Simpan media dari pesan WA lalu upload
const { downloadMediaMessage } = require('@whiskeysockets/baileys');
const FormData = require('form-data');
const fetch = require('node-fetch');

async function uploadFromWA(sock, msg) {
  const buffer = await downloadMediaMessage(msg, 'buffer', {});
  const mimeType = msg.message?.imageMessage?.mimetype || 'image/jpeg';
  const ext = mimeType.split('/')[1] || 'jpg';

  const form = new FormData();
  form.append('file', buffer, { filename: `media.${ext}`, contentType: mimeType });

  const res = await fetch('https://your-project.vercel.app/api/upload', {
    method: 'POST',
    body: form,
    headers: form.getHeaders(),
  });

  const data = await res.json();
  return data.url; // kirim balik ke WA
}

Contoh — axios

const axios = require('axios');
const FormData = require('form-data');
const fs = require('fs');

const form = new FormData();
form.append('file', fs.createReadStream('./video.mp4'));

const { data } = await axios.post(
  'https://your-project.vercel.app/api/upload',
  form,
  { headers: form.getHeaders() }
);
console.log(data.url);

  }

🚀 Cara Deploy ke Vercel

Ikuti langkah berikut untuk deploy gratis ke Vercel:

# 1. Clone / download project ini
# 2. Push ke GitHub
# 3. Import ke vercel.com → New Project

# 4. Di Vercel Dashboard → Storage → Create → Blob
#    Pilih plan "Hobby" (gratis 5GB)
#    Connect ke project kamu

# 5. (Opsional) Set environment variable:
UPLOAD_API_KEY = "rahasia-kamu-123"

# 6. Deploy → dapat domain .vercel.app gratis!
